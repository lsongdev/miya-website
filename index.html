01 / A clearer agent workspace

More than a chat window

Miya keeps sessions, agents, models, tools, and live activity in one native client. You can follow the useful details when you need them, without turning every conversation into a debug console.

02 / Workflow

From provider to useful work

Configure once, then move naturally between conversations and agents.

  1. 01

    Connect your model

    Add an OpenAI-compatible or Anthropic provider, choose a model, and create a profile.

  2. 02

    Choose an agent

    Use the embedded Miya runtime or connect an external ACP endpoint such as OpenCode.

  3. 03

    Watch work unfold

    See streaming messages, tool calls, plans, files, usage, and progress in a single timeline.

  4. 04

    Extend when needed

    Install skills, connect MCP servers, or bridge remote chat channels to the same agents.

03 / Capabilities

Built around open boundaries

Miya owns the desktop experience while ACP keeps the agent boundary portable.

ACP

Multiple agents, one session list

Built-in profiles and external ACP agents share the same navigation and conversation model.

Streaming

Readable live activity

Messages arrive fluidly while thoughts, tool calls, plans, files, and stop reasons keep their structure.

Tools

MCP and skills

Connect supported MCP servers and install reusable skills from dedicated marketplace views.

Channels

Agents beyond the desktop

Bridge Telegram, WeChat, Feishu, and other channels to a selected agent through Miya Channels.

Local

Your configuration, on your machine

Profiles, providers, skills, sessions, workspaces, and logs stay in the local Miya runtime directory.

Native

A focused desktop shell

Go and Wails provide a compact native application for macOS and Windows without a browser tab to manage.

05 / Open source

Inspect every layer

Miya Desktop is one part of a small, composable system. The desktop client, agent runtime, and channel gateway can be explored independently.

  • miya-desktop - native ACP client and settings UI
  • miya-agents - agent runtime, sessions, tools, MCP, and skills
